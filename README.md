# PRandM
This project will be about creating a repo, creating file in it, cloning it, creating 2 feature branches, merge feature 2 branch in feature 1 branch through pull request

# Steps
1. Create a repo
2. Create a file
3. Clone the repo
4. Create 2 feature branches (How to clone code from a feature branch instead of main)
    In such a way,
    a. Feature 1: Main code and more code added
    b. Feature 2: To pull code from the feature 1 code instead of main or master branch, add more code and merge feature 2 code into feature 1 code using pull request
